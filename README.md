# bot-tima — Ветка: politics-all-users

Бот следит за **всеми участниками** группового чата и грубо комментирует сообщения на политические темы.

## Поведение

- Реагирует на сообщения от **любого пользователя** в чате
- Срабатывает по ключевым словам: война, СВО, терроризм, названия стран и народов, международное право
- Не использует LLM-классификатор — только keyword-matching (быстро и точно)
- Отвечает грубым саркастичным комментарием с матом по теме сообщения

## Стек

- Node.js + TypeScript
- grammY (Telegram Bot API)
- Ollama (локальная LLM)
  - `qwen2.5:3b` — генерация саркастичного комментария
- Docker + Docker Compose

## Установка и запуск

```bash
cp .env.example .env
# Заполнить .env
docker compose up --build -d
```

## Конфигурация (.env)

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `OLLAMA_URL` | URL Ollama (по умолчанию `http://localhost:11434/v1`) |
| `WRITER_MODEL` | Модель для генерации (по умолчанию `qwen2.5:3b`) |

## Триггерные темы

- Война, СВО, боевые действия, мобилизация
- Терроризм, теракты, экстремизм, ИГИЛ
- Названия стран и народов (Россия, Украина, США, Китай и др.)
- Международное право, санкции, геноцид, трибунал

## Требования

- Отключить Privacy Mode в BotFather: `/mybots → Bot Settings → Group Privacy → Turn off`
